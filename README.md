# qbox-hosting

# Installation guide

## Get the start.sh...
cd /tmp; 

wget -qO start.sh https://raw.githubusercontent.com/josemorenoasix/qbox-hosting/master/server/start.sh; 

bash start.sh

## Edit install.cfg
nano install.cfg

## Execute as sudo install.sh
sudo bash install.sh

## Wait a few minutes... and done!
